# FIFA Player Management System

A Python command-line application to manage FIFA player records.

## Features
- 👤 Player Registration
- 📊 View All Player Scores
- 🔍 Search Player
- ✏️ Update Score
- 🏆 Top 2 Players by Score
- 💾 Data Persistence

## Installation

```bash
git clone https://github.com/YOUR-USERNAME/FIFA-Player-Management.git
cd FIFA-Player-Management
python fifa.py
```

## Menu Options

1. **Player Registration** - Register new players
2. **View All Players** - Display all players
3. **Search Player** - Find by name or ID
4. **Update Score** - Change player score
5. **Top 2 Players** - See top performers
6. **Exit** - Close program

## Requirements
- Python 3.6+
- No external libraries

## Data Storage
Data saved in FIFA.json
